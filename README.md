# 🚀 Career Quest — Job Application Tracker 

A modern, responsive, and gamified **Job Application Tracker** built with **React.js**, **React Router**, **Context API**, **Tailwind CSS**, and **Recharts**.

Career Quest helps job seekers organize, track, and analyze their job applications through a clean dashboard and an engaging user experience.

---

## ✨ Features

### 📋 Job Management
- ➕ Add new job applications
- ✏️ Edit existing applications
- 🗑️ Delete applications with confirmation
- 📅 Track application date
- 📝 Add personal notes
- 🔗 Save the original **Job Post Link**

### 📊 Dashboard
- View all job applications
- Search applications
- Filter by status
- Sort by newest or oldest
- Responsive dashboard

### 📈 Analytics
- Applications by Status
- Applications per Month
- Success Rate
- Interview Rate
- Offer Rate
- Rejection Rate
- Average Applications per Month

### 📅 Interview Scheduler
When a job reaches the **Interviewing** stage, you can save:
- Interview Date
- Interview Time
- Interview Type
- Interview Notes

The dashboard also displays upcoming interviews.

### 🚨 Smart CV Alert
If you've applied to many jobs without receiving an interview, Career Quest reminds you to:
- Review your CV
- Improve your Portfolio
- Tailor your applications

### ⭐ Favorites
Mark important opportunities as favorites and filter them easily.

### 🏢 Company Logos
Automatically display company logos with a graceful fallback when unavailable.

### 📍 Application Timeline
Track every application's progress from:
- Applied
- Recruiter Viewed
- Assessment
- Interview
- Offer

### 🎉 Offer Celebration
Receive a confetti celebration whenever an application reaches the **Offer** stage.

### 🎯 Gamification
- Quest Progress
- Achievement Badges
- Motivational Messages

### 🌙 User Experience
- Dark / Light Mode
- Glassmorphism UI
- Responsive Design
- Toast Notifications
- Loading Skeletons
- Empty State Illustration

### 💾 Local Storage
All data is stored locally in your browser.

### 📤 Import / Export
- Export applications as JSON
- Import applications from JSON

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Doha-2004/career-quest-new.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
